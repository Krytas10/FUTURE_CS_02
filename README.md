Task 2: Phishing Email Detection & Awareness System

📌 Project Overview
This repository contains a professional Phishing Email Detection & Awareness Report analyzing three real-world phishing email samples representing distinct attack categories: credential theft, personal data harvesting, and Business Email Compromise (BEC). The primary objective of this project was to identify common phishing indicators, classify email threats by risk level, and produce clear, actionable awareness guidelines to help users recognize and avoid phishing attacks.

🛠️ Tools & Technologies Used

* MXToolbox: Email header, SPF, and DMARC record analysis.
* Whois: Domain registration and ownership investigation.
* VirusTotal: URL and domain reputation analysis against multiple security vendors.
* URLScan.io: Investigation of suspicious URLs and website behavior.
* Browser Developer Tools: Manual inspection of suspicious links and webpage elements.
* Microsoft Word: Professional document design and reporting layout.

🔍 Key Findings Summary
The analysis successfully identified and classified 3 distinct phishing attack samples:

1. PayPal Account Limitation Scam (Critical): Spoofed sender domain (`paypal-accounts.com`) combined with urgency tactics to harvest user login credentials.
2. FedEx Delivery Scam (High): Lookalike branding and a fake address verification request used to harvest personal data via a disposable phishing domain.
3. CEO Fraud Email (Critical): Executive impersonation and social engineering used to bypass financial approval processes and solicit fraudulent gift voucher purchases, with no reliance on technical infrastructure.
